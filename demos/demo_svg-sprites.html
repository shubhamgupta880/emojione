Noteworthy Problems with using SVG sprites:

To get these working in Internet Explorer 8-11 you need to add the following meta tag and the SVG4Everybody polyfill (available here). 

<meta http-equiv="X-UA-Compatible" content="IE=Edge">
<script src="svg4everybody.ie8.min.js"></script>

Additionally, loading in the SVG sprites cross domain does not work, and cannot be included on our CDN. SVG sprite file needs to be hosted on the same domain its requested from.
