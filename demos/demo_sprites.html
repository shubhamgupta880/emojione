Emoji One PNG Sprite Example

Class names are .emojione- suffixed with appropriate unicode character code (all caps).

The code for the each following sections is identical, only changing the font-size applied to the container. You can also apply the font-size directly to [class*=emojione-].

It's highly recommended you keep the computed height and width values of the emoji at hard pixel values to ensure best display in firefox.

Please note that our PNG sprite set is still being tested, and as such is not yet available as an option in our translation scripts. 

      
<div style="font-size: 16px">
    <div class="emojione-1F352"></div>
    <div class="emojione-1F353"></div>
    <div class="emojione-1F354"></div>
    <div class="emojione-1F355"></div>
    <div class="emojione-1F356"></div>
    <div class="emojione-1F357"></div>
</div>
