<script type="text/javascript">
    $(document).ready(function() {
       $("#demo5-input").on('keyup change input',function(e) {
           var source = $('#demo5-input').val();
           var preview = emojione.toImage(source);
           $('#demo5-output').html(preview);
        });

</script>

Demo #6

Automatically convert Unicode characters to shortnames upon form submission

Input:





<script type="text/javascript">
    $("#demo6-form").on('submit',function(e) {
        var input = $('#demo6-input').val();
        var replaced = emojione.toShort(input);
        $('#demo6-input').val(replaced);

        // prevent form submission
        // normally you wouldn't do this but we don't want your page to reload for this demo
        e.preventDefault();
    });
</script>

Demo #7

Stick a class of emojione-convert on any HTML element and automatically convert shortnames after page load

Welcome to the Emoji One :snail: demo! I hope you like we've put together here for you. :thumbsup: :smile: 
<div class"emojione-convert">
    Welcome to the Emoji One :snail: demo!
    I hope you like what we've put together here for you. :thumbsup: :smile:
</div>

<script type="text/javascript">
     $(document).ready(function() {
        $(".emojione-convert").each(function() {
            var original = $(this).html();
            var converted = emojione.toImage(original);
            $(this).html(converted);
        });
    });
</script>
